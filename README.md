Instalação
------------
Front end ITSS 

#### Download:

Download from [Github releases](https://github.com/mirandati7/front-itss).

#### Using The Command Line:

__Via Git__
- Clone to your machine
```
git clone https://github.com/mirandati7/front-itss
```

__Via NPM__
```bash
npm install 
```
